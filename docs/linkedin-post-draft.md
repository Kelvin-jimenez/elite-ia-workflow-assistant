## Draft Post

I have started documenting a new project on GitHub: Elite IA Workflow Assistant.

The goal of this project is to build an AI-powered workflow assistant that helps manage client requests for small businesses and digital service providers.

The first MVP focuses on a simple but useful process:

- Receive client requests through a form
- Store the information in Google Sheets
- Notify the responsible person
- Classify the request using AI
- Generate a suggested response
- Keep human approval before sending anything

This project combines several areas I am currently learning and applying:

- Workflow automation
- AI and LLMs
- Product thinking
- Business process automation
- GitHub documentation
- Security and privacy principles

The objective is not just to learn tools, but to build a practical solution that could solve a real business problem: reducing repetitive manual work when managing client requests.
