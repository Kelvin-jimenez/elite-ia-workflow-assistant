# Basic Client Request Workflow

## Product Name

Elite IA Workflow Assistant

## Workflow Objective

This workflow describes the first automation flow for the MVP.

The goal is to receive a client request from a form, store the information in Google Sheets and notify the responsible person.

## Workflow Version

Version: 1.0  
Status: Draft

## Tools

The first version of the workflow will use:

- Google Forms or Tally
- n8n
- Google Sheets
- Email notification

## Workflow Trigger

The workflow starts when a client submits a request form.

Trigger:

```text
New form submission
